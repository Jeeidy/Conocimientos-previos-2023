Etiquetas Semánticas: Header: Define el encabezado de una página web o sección. Nav: Define una sección con enlaces de navegación. Section: Define una sección dentro de una página web. Article: Define un bloque de contenido autónomo que puede existir en cualquier contexto. Puede tener su propio encabezado, pie de página, secciones... Útil para una lista de publicaciones del blog. Aside: Define un bloque de contenido que está relacionado con el contenido principal. Generalmente se muestra como una barra lateral. Footer: Define el pie de página de una página o sección web. Etiquetas de Texto: < p >: Para escribir párrafos < h1 >: Títulos Etiquetas de Agrupación: < div >: Capa o división utilizado para agrupar varias etiquetas HTML. < p >: Define un párrafo de texto. < main >: Contenedor para englobar la parte principal de la página. < hr >: Indica una separación temática del texto. < ol >: Crea una lista numerada (con orden). < ul >: Crea una lista donde el orden no importa. < li >: Contiene uno de los ítems de una lista numerada o sin numerar. < dl >: Crea una lista de definiciones. Etiquetas Multimedia: < img >: Etiquetas de Tablas: < table >: Etiqueta contenedora que tendrá en su interior toda la tabla. < tr >: Table Row. Etiqueta contenedora de cada fila de la tabla. < td >: Table Data. Cada una de las celdas de la tabla. < th >: Table Header. Cada una de las celdas de cabecera de la tabla. Etiquetas de formularios: < form >: Define un formulario < label >: Representa el título de un elemento de control de un formulario. < input >: Se usa para crear controles interactivos que reciben datos del usuario. < button >: Representa un botón. < select >: Representa un elemento de control que permite la selección entre un conjunto de opciones < option > < option >: Representa una opción en un elemento < select > o < datalist >. < datalist >: Se utiliza junto con la etiqueta < input > para proporcionar una lista de opciones < option > que ayudan a los usuarios a seleccionar un valor mientras escriben en un campo de entrada < input >.
